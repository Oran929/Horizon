## [SGLang v0.5.15 将 GLM-5.2 在 Blackwell 上的速度提升至 500+ tok/s](https://github.com/sgl-project/sglang/releases/tag/v0.5.15) ⭐️ 8.0/10

SGLang v0.5.15 为 Blackwell 硬件上的 GLM-5.2 NVFP4 提供了重大性能优化，在 8 块 B300 GPU 上实现了每用户每秒超过 500 个 token。关键改进包括默认启用 Spec V2 推测解码、IndexShare MTP、TopK V2 以及索引器前序融合。 此版本显著提升了大语言模型（尤其是 GLM-5.2）的服务效率，使在 NVIDIA 最新的 Blackwell 架构上实现高吞吐推理更加可行。这些优化降低了延迟和成本，有利于大规模 LLM 服务的生产部署。 Spec V2 通过零开销调度和融合元数据操作实现了 11% 的端到端 TPS 提升。IndexShare MTP 通过在 draft 步骤间复用索引器 top-k，在长上下文下将 draft 步骤成本降低高达 1.9 倍。TopK V2 将 top-k 选择与页表变换融合，支持运行时 k 值高达 2048。

github · Fridge003 · 7月10日 22:58

**背景**: SGLang 是一个开源的大语言模型服务框架，用于优化推理吞吐量和延迟。推测解码是一种技术，通过一个小型 draft 模型预测多个 token，再由目标模型验证，从而提升速度。NVFP4 是 NVIDIA 为 Blackwell GPU 开发的 4 位浮点量化格式，用于高效推理。

## [快手与浙大开源 GUI 智能体 MobileForge](https://news.google.com/rss/articles/CBMiVkFVX3lxTE84RkhUUzFDVkdiX2huWGVLa2owdmJDbms3MFpiS0hJRlpUQkJ0aDNFWmpzTm8tZDh2RHFLQUx0SHVxNGROMkVJR0dENXBXWTc5ZGxwMkhR?oc=5) ⭐️ 7.0/10

快手与浙江大学开源了 MobileForge，这是一个利用数据飞轮无需人工标注即可训练的 GUI 智能体。该系统通过分层反馈引导的策略优化，自主适应移动 GUI 任务。 这降低了为 GUI 智能体收集标注数据的成本和精力，加速了其在移动自动化和无障碍领域的部署。它代表了向能够从真实世界交互中学习的自我改进 AI 智能体迈出的重要一步。 MobileForge 使用分层回滚反馈和提示上下文化的策略更新，无需人工编写的任务、演示或奖励标签。代码已在 GitHub 的 kwai 组织下开源。

google_news · 投资界 · 7月10日 01:09

**背景**: GUI 智能体是像人类一样与图形用户界面交互的 AI 系统，执行点击按钮或填写表单等任务。传统训练需要对截图和动作序列进行大量人工标注，耗时且昂贵。数据飞轮概念创建了一个自我强化的循环，智能体通过从自身交互中学习来改进，减少了对人工输入的需求。
